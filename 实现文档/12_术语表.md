# 术语表

| 术语 | 中文解释 | 在本实验中的含义 |
| --- | --- | --- |
| `seq_len` | 输入序列长度 | 用过去多少小时作为输入，第一版为 96 小时 |
| `pred_len` | 预测序列长度 | 预测未来多少小时，第一版为 336 小时 |
| `horizon` | 预测步长 | 未来第几天，范围 1-14 |
| `peak_value` | 峰值 | 一个自然日内某列的最大值 |
| `peak_hour` | 峰值出现小时 | 峰值第一次出现的小时，范围 0-23 |
| `baseline` | 规则基线 | 不使用机器学习的简单预测规则 |
| `residual` | 残差 | 真实值减去基线预测值 |
| `residual learning` | 残差学习 | 模型预测残差，再加回基线 |
| `LightGBM` | 轻量梯度提升树模型 | 第一版主模型之一，适合表格特征 |
| `XGBoost` | 极端梯度提升树模型 | 第一版主模型之一，用于与 LightGBM 对照 |
| `MAE` | 平均绝对误差 | `abs(预测值-真实值)` 的平均 |
| `RMSE` | 均方根误差 | 对大误差更敏感的误差指标 |
| `sMAPE` | 对称平均绝对百分比误差 | 用百分比形式比较误差 |
| `wavelet transform` | 小波变换 | 将序列分解为不同尺度成分的方法 |
| `DWT` | 离散小波变换 | 第二版计划使用的小波变换形式 |
| `high frequency energy` | 高频能量 | 反映局部扰动、尖峰或噪声强度 |
| `low frequency trend` | 低频趋势 | 反映较慢变化的趋势成分 |
| `denoising` | 去噪 | 过滤小幅噪声，保留主要形态 |
| `turning point` | 拐点 | 趋势方向发生反转的点 |
| `prominence` | 峰谷显著性 | 衡量一个峰或谷相对周围有多突出 |
| `MAD` | 中位数绝对偏差 | 一种稳健的局部波动度量 |
| `feature` | 特征 | 输入给模型的数值列 |
| `label` | 标签 | 模型要预测的真实目标 |
| `train` | 训练集 | 用于拟合模型的数据 |
| `validation` | 验证集 | 用于调参和早停的数据 |
| `test` | 测试集 | 用于最终评估的数据 |
| `early stopping` | 早停 | 验证集长期不提升时停止训练 |
| `feature importance` | 特征重要性 | 衡量模型使用某个特征的程度 |
| `data leakage` | 信息泄露 | 训练特征中错误使用了未来信息 |
| `ablation study` | 消融实验 | 只改变一个因素，观察它是否有贡献 |
