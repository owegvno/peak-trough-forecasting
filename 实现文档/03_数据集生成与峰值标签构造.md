# 数据集生成与峰值标签构造

## 1. 目标

本文件说明如何从 `ETTh1.csv` 生成任务 1 所需的数据集。

生成目标：

```text
过去 seq_len 小时的 7 列历史特征
未来 pred_len 小时内按自然日统计的 peak_value 和 peak_hour 标签
```

第一版默认：

```text
seq_len = 96h
pred_len = 336h
未来周期数 = 14 天
目标列 = HUFL,HULL,MUFL,MULL,LUFL,LULL,OT
```

## 2. 输入文件

```text
ETTh1.csv
```

必须包含字段：

```text
date, HUFL, HULL, MUFL, MULL, LUFL, LULL, OT
```

## 3. 数据检查

读取后先检查：

| 检查项 | 要求 |
| --- | --- |
| 时间列 | 可解析为时间 |
| 时间间隔 | 相邻记录应为 1 小时 |
| 数值列 | 7 个目标列均可转为数值 |
| 缺失值 | 第一版不接受目标列缺失 |
| 重复时间 | 不允许重复时间戳 |

如果发现缺失小时，第一版不插值，应先记录问题并停止生成。

## 4. 生成完整自然日表

按 `date` 提取日期部分，统计每个日期是否有 24 个小时：

```text
00:00, 01:00, ..., 23:00
```

只有完整自然日进入周期表。不完整日期直接丢弃。

对于当前数据，末尾不满 24 小时的日期不参与任何样本。

## 5. 峰值标签构造

对每个完整自然日、每个变量计算：

```text
peak_value = 当天最大值
peak_hour = 当天最大值第一次出现的小时
```

示例：

```text
如果 OT 在 10:00 和 15:00 都达到当天最大值
peak_hour = 10
```

每日标签表建议字段：

```text
date
HUFL_peak_value, HUFL_peak_hour
HULL_peak_value, HULL_peak_hour
MUFL_peak_value, MUFL_peak_hour
MULL_peak_value, MULL_peak_hour
LUFL_peak_value, LUFL_peak_hour
LULL_peak_value, LULL_peak_hour
OT_peak_value, OT_peak_hour
```

建议输出：

```text
数据集/日周期峰值标签.csv
```

## 6. 样本生成方式

以预测起点日 `D` 为样本锚点。

输入窗口：

```text
D-4, D-3, D-2, D-1
```

预测窗口：

```text
D, D+1, ..., D+13
```

要求输入窗口和预测窗口都必须由完整自然日组成。否则不生成样本。

## 7. 样本表结构

建议生成长表，而不是一个极宽表。

长表中每一行表示：

```text
某个样本锚点
某个目标变量
某个未来 horizon
```

字段建议：

```text
sample_id
forecast_start_date
target_col
horizon
target_date
baseline_peak
target_peak_value
target_peak_residual
target_peak_hour
特征列...
split
```

其中：

```text
horizon = 1 表示未来第 1 天
horizon = 14 表示未来第 14 天
```

## 8. 切分策略

采用 ETTH1 常见切分：

```text
训练集：前 12 个月
验证集：后续 4 个月
测试集：后续 4 个月
```

样本归属建议按目标窗口判断：

```text
如果预测窗口完整落在训练区间，则为训练样本
如果预测窗口完整落在验证区间，则为验证样本
如果预测窗口完整落在测试区间，则为测试样本
```

这样可以避免训练样本的预测目标跨入验证或测试区间。

## 9. 防止信息泄露

禁止在输入特征中使用：

```text
预测起点日 D 及之后的真实值
未来 14 天的峰值、均值、最大值
由未来窗口计算出的拐点、波峰或小波特征
```

允许使用：

```text
D 之前的历史小时值
D 之前的历史日峰值标签
D 的日历信息
未来 horizon 的编号
未来目标日的 weekday、month、day_of_year 等已知日历信息
```

## 10. 建议输出文件

```text
数据集/完整小时数据.csv
数据集/完整自然日清单.csv
数据集/日周期峰值标签.csv
数据集/峰值预测样本_seq96_pred336.csv
数据集/峰值预测样本_seq96_pred336_训练.csv
数据集/峰值预测样本_seq96_pred336_验证.csv
数据集/峰值预测样本_seq96_pred336_测试.csv
数据集/数据集生成报告.md
```

## 11. 数据集生成报告

报告至少包含：

| 项目 | 内容 |
| --- | --- |
| 原始行数 | `ETTh1.csv` 原始小时行数 |
| 完整自然日数量 | 可用于建模的完整天数 |
| 丢弃日期 | 不完整日期列表 |
| 样本数量 | train/val/test 各自样本数 |
| 目标列 | 7 个变量 |
| seq_len | 当前输入长度 |
| pred_len | 当前预测长度 |
| 缺失值检查 | 是否存在缺失 |
| 泄露检查 | 是否通过 |
