# 第二版小波特征实验

## 1. 启动条件

第二版小波实验必须在第一版完成后启动。

第一版包括：

```text
规则基线
统计特征
残差学习
LightGBM
XGBoost
```

第二版目标不是替代第一版，而是在第一版基础上增加小波特征，验证是否带来增益。

## 2. 小波特征目标

小波特征用于捕捉：

```text
低频趋势变化
高频扰动增强
局部尖峰风险
去噪后的峰值形态
多尺度波动结构
```

它主要服务 `peak_value`，也可能辅助 `peak_hour`。

## 3. 小波处理对象

对 7 个变量分别处理：

```text
HUFL, HULL, MUFL, MULL, LUFL, LULL, OT
```

每个样本只能使用预测起点之前的输入窗口做小波分解。

第一版窗口：

```text
过去 96 小时
```

如果后续 `seq_len` 增长，小波窗口也随之增长。

## 4. 推荐小波方法

第一版小波增强建议使用离散小波变换。

推荐小波基：

```text
db4
```

`db4` 是 Daubechies 小波的一种，常用于平滑但保留局部变化的信号处理。

推荐分解层数：

```text
根据 seq_len 自动选择，不超过可稳定分解的最大层数
```

对 `seq_len=96h`，不宜分解太深，避免高层系数不稳定。

## 5. 小波特征清单

对每个变量计算：

```text
wavelet_low_last：低频重构序列最后值
wavelet_low_slope：低频趋势斜率
wavelet_high_energy：高频细节能量
wavelet_high_ratio：高频能量占总能量比例
wavelet_denoised_max：去噪序列最大值
wavelet_denoised_last：去噪序列最后值
wavelet_detail_abs_mean：细节系数绝对值均值
wavelet_detail_abs_max：细节系数绝对值最大值
wavelet_recent_high_energy_change：最近 24 小时高频能量变化
```

## 6. 加入方式

第二版特征表应在第一版特征表基础上增加小波列。

实验对比必须保持：

```text
同一训练验证测试切分
同一规则基线
同一模型参数初始策略
同一评估指标
```

否则无法判断提升是否来自小波。

## 7. 消融实验

至少比较：

| 实验 | 特征 | 模型 |
| --- | --- | --- |
| 第一版 | 统计特征，无小波 | LightGBM/XGBoost |
| 第二版 | 统计特征加小波 | LightGBM/XGBoost |

可选进一步消融：

```text
只加低频特征
只加高频特征
只加去噪峰值特征
```

## 8. 成功判定

小波特征有价值的表现：

```text
peak_value MAE 下降
远 horizon 表现改善
极端峰值预测改善
预测峰值分布不再过度塌缩
特征重要性中小波特征有稳定贡献
```

## 9. 失败判定

以下情况说明小波不适合当前版本：

```text
测试集 MAE 没有提升
训练集提升但验证/测试下降
小波特征重要性极低
预测分布更不稳定
不同变量提升不一致且整体无收益
```

如果失败，保留第一版作为主线，不继续堆叠更多小波特征。

## 10. 信息泄露注意事项

小波特征只能从输入窗口计算。

禁止：

```text
对完整数据集一次性小波分解后再切样本
使用预测窗口内的数据计算小波系数
用未来 14 天去噪峰值作为特征
```

正确方式：

```text
每个样本单独取预测起点之前的 seq_len 小时
只对这段历史做小波分解
生成该样本的小波特征
```
